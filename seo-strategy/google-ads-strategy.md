## Conversion Tracking — What to Set Up

### Required conversion events (set up BEFORE launching):

1. **Phone call from ad** — Google auto-tracks calls from call extensions. Enable in Google Ads → Goals → Calls from ads. Set minimum 60-second call as conversion.

2. **Website contact form submission** — Install Google tag on your site. Create a goal that fires when the thank-you page URL is reached (e.g., /thank-you). If using Netlify Forms, add conversion tracking to the form submission redirect.

3. **Phone call from website** — Use Google forwarding number on your landing page. Tracks calls that start from your site after someone clicks an ad.

4. **Click-to-call button** — If your mobile landing page has a tap-to-call button, tag it as a conversion event.

## Step-by-Step Google Ads Setup Instructions

### Step 1: Create Google Ads Account
1. Go to ads.google.com
2. Sign in with your business Gmail (crcp183@gmail.com or create ads@totalpropertysolution.net)
3. Select "Switch to Expert Mode" when prompted (IMPORTANT — do not use Smart campaigns)
4. Enter billing info — set monthly budget cap of $750 (10% buffer)

### Step 2: Link Google Business Profile
1. In Google Ads → Tools → Linked Accounts → Google Business Profile
2. Connect totalpropertysolution.net profile
3. This enables location extensions automatically

### Step 3: Set Up Conversion Tracking
1. Go to Tools → Measurement → Conversions
2. Click + New Conversion Action
3. Choose "Website" → enter totalpropertysolution.net
4. Create goal for "Contact form submission" (thank-you page URL)
5. Create goal for "Phone call" (set 60 second minimum)
6. Install the Google tag on your website (give this code to your web developer)

### Step 4: Create Campaign 1 (Commercial Cleaning)
1. Click + New Campaign
2. Goal: Leads
3. Campaign type: Search
4. Uncheck: "Search Network" and "Display Network" partners for now
5. Name: "TPS Pro — Commercial Cleaning Albany"
6. Bidding: Maximize Conversions (switch to Target CPA $80 after 30 conversions)
7. Daily budget: $10
8. Location: Albany NY + 30 mile radius (manually verify the cities listed above are included)
9. Language: English
10. Ad schedule: Mon–Fri 7am–7pm, Sat 8am–3pm

### Step 5: Create Ad Groups in Campaign 1
1. Click + New Ad Group within the campaign
2. Name: "Albany Commercial Cleaning — Core"
3. Add keywords from Ad Group 1 list (phrase and exact match)
4. Build first RSA ad using headlines and descriptions from Ad Variation 1
5. Repeat for Ad Groups 2 and 3

### Step 6: Add Negative Keywords
1. In Campaign 1 → Keywords → Negative Keywords
2. Add all negatives from the list above as broad match
3. Consider creating a shared negative keyword list (Tools → Shared Library → Negative Keyword Lists) — name it "Non-Commercial" and apply to all 3 campaigns

### Step 7: Add Ad Extensions/Assets
1. Campaign level → Assets
2. Add Call extension: (518) 948-7156
3. Add Sitelinks (4 minimum)
4. Add Callout extensions
5. Add Structured Snippet
6. Location extension auto-populates from GBP link

### Step 8: Repeat for Campaigns 2 and 3
Follow same process. Different ad groups, different keywords, different landing pages.

### Step 9: Launch and First-Week Monitoring
- Check daily: impressions, clicks, CTR
- Watch for irrelevant search terms in "Search Terms" report (add negatives immediately)
- Do NOT change bids or budgets in the first 7 days — let Google learn

### Step 10: Week 2+ Optimization
- Pause keywords with 50+ clicks and zero conversions
- Increase bids on top-converting keywords
- A/B test ad variations (let each run 2+ weeks before judging)
- Add converting search queries as exact match keywords
